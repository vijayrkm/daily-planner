Welcome to Daily Tasks
One-time setup · takes about 5 minutes
1
Create the Google Sheet
In Google Drive, create a new Sheet named Daily Critical Activities.
2
Open Apps Script
Inside the Sheet → Extensions → Apps Script. Delete existing code and paste in the Code.gs content. Save.
3
Deploy as Web App
Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone → Deploy → authorise → copy the URL.
4
Paste your Web App URL
Tasks are stored in your Google Sheet. The URL is saved only on this device.
