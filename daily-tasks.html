Welcome to Daily Tasks
One-time setup · takes about 5 minutes
1
Create the Google Sheet
Go to Google Drive → SELF OR COMM / 02 G_CMGT. Create a new Google Sheet and name it Daily Critical Activities.
2
Open Apps Script
Inside the Sheet, click Extensions → Apps Script. Delete any existing code in the editor.
3
Paste the backend code
Open Code.gs (provided with this file). Copy everything and paste into the Apps Script editor. Press Cmd+S to save.
4
Deploy as Web App
Click Deploy → New deployment. Choose:
· Type: Web app
· Execute as: Me
· Who has access: Anyone
Click Deploy, authorise when prompted, then copy the Web App URL.
5
Paste your Web App URL here
Paste the URL you copied in step 4:
Your tasks live in your Google Sheet in Drive.
The URL is saved only on this device.
